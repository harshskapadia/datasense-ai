# SupportSmart AI

SupportSmart AI is a hybrid AI customer service chatbot designed for large e-commerce customer support environments.

The project is based on the idea that AI chatbots should not only provide fast responses, but should also support customer trust, response accuracy, ethical reliability, and human escalation when needed.

## Features

- AI-powered customer service chatbot
- Handles simple customer support queries
- Classifies customer messages by complexity
- Detects sensitive or complex issues
- Escalates refund, payment, privacy, security, legal, or complaint-related issues to a human support agent
- Maintains conversation history
- Logs customer queries
- Provides a chatbot performance dashboard
- Allows downloading conversation and escalation logs

## Use Cases

SupportSmart AI can be used for:

- Product enquiries
- Delivery questions
- Refund and return support
- Customer complaints
- Payment-related concerns
- Privacy or account security issues
- Human escalation workflows

## Tech Stack

- Python
- Streamlit
- OpenAI API
- pandas
